# Two layers, specifics set on the land

## Context
What we measure to know the work is succeeding. The model separates plain project counts from ecological recovery (see [indicators](../../template/indicators/00-overview.md)).

## Decision (deferred)
Adopt the model's two-layer approach — countable project facts for the public view, and ecological recovery measured to test methods. The specific indicators and baselines wait until we are on a site and can set them against real conditions.

## Why
Indicators chosen before there is land to measure would be guesses; the model's shape is enough to commit to now.

## Constraints
No site yet, so no baseline.

## Revisit when
A site is secured and the first baseline survey can be designed.
