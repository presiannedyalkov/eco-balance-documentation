## Coordination Best Practices

### When Running Multiple Personas
1. **Start with Janitor** - Clean organization enables all other work
2. **Run Auditor early** - Identify gaps before filling them
3. **Run Security Auditor before publishing** - Prevent sensitive data exposure
4. **Run Quality Inspector last** - Polish after content is complete
5. **Run Stakeholder Advocate before major releases** - Ensure messaging resonates

### High-Value Persona Combinations
1. **Knowledge Synthesizer + Curator** - Research-backed content enhancement
2. **Strategic Advisor + Implementation Coach** - Reality-tested execution plans
3. **Security Auditor + Curator** - Safe, compliant public content
4. **Stakeholder Advocate + Curator** - Audience-aligned messaging
5. **Auditor + All Personas** - Systematic completeness verification

### Conflict Resolution Priority
When personas suggest conflicting changes:
1. **Security Auditor > All** - Security and privacy take precedence
2. **Strategic Advisor > Implementation Coach** - Strategy guides execution
3. **Stakeholder Advocate > Curator** - Audience needs drive messaging
4. **Auditor > Curator** - Completeness before enhancement
5. **Quality Inspector runs last** - Polish after all content decisions made

**Goal**: Prepare v2.0.0 for public release
**Persona Sequence**:
1. **Janitor** - Organize all files properly
2. **Auditor** - Verify nothing is missing

4. **Stakeholder Advocate** - Verify messaging for all audiences
