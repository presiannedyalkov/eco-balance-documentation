## üéØ Core Philosophy from User

**Key Principles**:
1. **Nothing is concrete yet** - everything is organic, evolving based on opportunities
2. **Multiple options everywhere** - minimum 3 options per decision, or "d) Other/TBD"
3. **Flexibility and agility** - ready to go anywhere, work with anyone under right conditions
4. **Security first** - basic health, financial security are non-negotiable constraints
5. **Path creation** - where no path exists, we create it
6. **Premature precision avoided** - detailed specs (like solar calculations) are pointless until context is clear
7. **Vision-driven, detail-adaptive** - clear on what (restore ecosystems) but flexible on how

**User's Words**:
> "We want to do eco restoration but we want enough basic security, health, financial etc. We're willing to work with anyone who can offer us a path. Where no path exists, we create it. Same as land, same as money, same as concrete plants. Nothing is concrete currently, so there is no need to put a pin on everything - it is happening organically."
