### Q: Should I use the new personas or the original 6?
**A**: Use **both**! The original 6 handle maintenance (organization, links, formatting), while the new 5 handle strategy, research, security, execution, and stakeholder concerns. They complement each other.

### Q: Which persona should I run first?

### Q: How often should I run these personas?
**A**:

- **Implementation Coach**: When planning new phases
- **Knowledge Synthesizer**: Monthly or when research accumulates
- **Stakeholder Advocate**: Before every external communication

**A**: Yes! Use the workflow patterns in this guide. Example:
